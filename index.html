Echtzeit

A library-agnostic enterprise and highly optimized toolset for pub/sub messaging between web clients built to scale.

Echtzeit is a highly optimized toolset for pub/sub messaging between web clients built to scale.

This is an enterprise fork of Faye, please see http://faye.jcoglan.com for further information on the general purpose toolset with Ruby/Rack support.
